Tools

JavaScript

covjson-reader

A CovJSON reader supporting subsetting and lazy loading.

leaflet-coverage

A Leaflet plugin for visualizing CovJSON and other coverage data.

webworldwind-covjson (WIP)

A Web World Wind plugin for visualizing CovJSON and other coverage data.

Java

ncWMS

A WMS server with CovJSON as output format.

Python

pycovjson (WIP)

Transforms common scientific data formats like netCDF into CovJSON.

How to add your tool

Just submit a pull request for the tools.yaml file on GitHub, and once accepted, your tool will appear here immediately. Create an issue if your tool is written in a language other than the ones above.
